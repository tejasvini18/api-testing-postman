# API Testing Project (Postman)

## Project Overview
This project demonstrates API testing using Postman.

## APIs Covered
- Get All Users (GET)
- Get Single User (GET)
- Create User (POST)
- Update User (PUT)
- Delete User (DELETE)

## Features
- CRUD Operations testing
- Environment variables (base_url)
- Test scripts for validation
- Positive and negative testing

## Tools Used
- Postman
- JavaScript

## How to Run
1. Import collection JSON into Postman
2. Select environment
3. Click Send to execute APIs
